Hardcore Mode

Hardcore is a plugin for pocketmine that aims to implement a typical java vanilla game mode in a certain world, it will not copy 100% aspects of the mode, but it will also have many more features that can be customized in config.yml, this plugin was created by Henry Thierry [HenryDM]
